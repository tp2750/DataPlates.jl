Plate skeleton

Singel plate

Generate a DataFrame corresponding to a 96 well plate:

julia> DataFrame(DataPlate("Plate1"))

Sorting

The plate is sorted "by column". It is easy to sort it by row;

julia> sort(DataFrame(DataPlate("Plate1")))
96×4 DataFrame

Supported formats

Other known plate formats are 6, 24, 96, 384 well plates. Support for 1536 may come if needed.

Multiple plates

To make multiple plates, broadcast over the plate name (as possibly the barcode). Example:

julia> DataFrame(DataPlate.(["P1", "P2"], ["A", "B"],6))
12×4 DataFrame
 Row │ platename  barcode  well    well006 
     │ String     String   String  String  
─────┼─────────────────────────────────────
   1 │ P1         A        A01     A01
   2 │ P1         A        B01     B01
   3 │ P1         A        A02     A02
